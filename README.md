## Introduction
**OptimusTech** is a website dedicated to job openings at a technology company called OptimusTech. The website features a clear and attractive interface, allowing potential candidates to explore available opportunities and learn more about the company's culture and values.

## Installation

### Prerequisites
- Web browser
- Code editor like Visual Studio Code (optional for local development)

### Steps to run
1. Clone the repository:
   ```bash
   git clone https://github.com/victorhubarb/optimus-tech.git
   cd optimus-tech
   ```
2. Open the `index.html` file in your browser to view the page.

## Usage
The **OptimusTech** page displays job openings, organized by department and presented in detailed cards, allowing users to learn more about each position. The site also highlights the companyâ€™s commitment to remote work and includes employee testimonials.

## Features
- **Intuitive Navigation**: Navigation menu with links to the company's main sections.
- **Job Vacancies**: List of vacancies by department, with information about the position, type of contract and salary range.
- **Employee Testimonials**: Section with employee feedback about the work environment.
- **Application Form**: Sign up to receive updates on new vacancies.
